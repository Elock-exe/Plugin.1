InfectedTag

Version: 1.0  API: 1.20 By ElockExe : Neralix

Plugin Minecraft pour organiser des événements InfectedTag avec gestion des infections, effets spéciaux et zones.
Commandes principales
/joininfected – Rejoindre l'événement
/leaveinfected – Quitter l'événement
/infecter <joueur> – Infecter un joueur
/desinfecter <joueur> – Désinfecter un joueur
/desinfectall – Rendre humains tous les infectés
/infectedlist – Lister les participants et leur statut
Zones
/setzone1 à /setzone12 – Définir les points des zones d’événements
/setsafezone1 / /setsafezone2 – Définir la SafeZone
Fonctionnalités
Infection automatique au contact
Effets spéciaux pour les infectés : cécité, confusion, résistance
Respawn aléatoire dans les zones (hors SafeZone)
Particules visibles autour des zones et SafeZone
Installation
Placer InfectedTag.jar dans le dossier plugins
Redémarrer le serveur
Configurer les zones avec les commandes /setzoneX et /setsafezoneX
